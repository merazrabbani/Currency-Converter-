💱 Terminal Currency Converter

A simple yet powerful terminal-based Currency Converter built with Python. This project allows users to convert between different currencies through an interactive command-line interface.

✨ Features

- Convert between multiple currencies
- Easy-to-use menu-driven interface
- Input validation and error handling
- Conversion history tracking
- Timestamped conversion records
- Beginner-friendly code structure
- Modular and maintainable design

🛠 Technologies Used

- Python 3
- Dictionaries
- Functions
- Loops
- Exception Handling
- Datetime Module

🚀 How to Run

1. Clone the repository:

git clone https://github.com/your-username/currency-converter.git

2. Open the project folder:

cd currency-converter

3. Run the program:

python main.py
