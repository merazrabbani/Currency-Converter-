here# 💱 Terminal Currency Converter

A feature-rich and beginner-friendly **Currency Converter** built with **Python** that runs entirely in the terminal. This application allows users to convert currencies, track conversion history, and explore core programming concepts through a practical real-world project.

Whether you're learning Python or looking for a simple finance-related application, this project provides a clean and interactive command-line experience.

---

## ✨ Features

* 💱 Convert between multiple currencies
* 🖥️ Interactive menu-driven interface
* ✅ Input validation and error handling
* 📜 Conversion history tracking
* 🕒 Timestamped conversion records
* 🔄 Real-time conversion calculations (using predefined rates)
* 📚 Beginner-friendly and well-structured code
* 🧩 Modular and maintainable design

---

## 🛠️ Built With

* **Python 3**
* Functions

## 🚀 Getting Started

### Prerequisites

Make sure you have **Python 3.x** installed on your system.

### Installation

Clone this repository:

```bash
git clone https://github.com/your-username/currency-converter.git
```

Navigate to the project directory:

```bash
cd currency-converter
```

Run the application:

```bash
python main.py
```
